Melvoredit

Import and export save data for http://melvoridle.com/

1. Import
Character
Currencies
Skills
Skill Level XP Mastery Pool XP Action Mastery
Level is derived from XP (editing one updates the other). Combat skills have no mastery. "Max all" sets every action you've trained in that skill to mastery level 99. Equipment and other details live in the Advanced editor below.
Bank
Edit a quantity, or press × to remove an item. You can only add items already known to your save (their IDs must be in the save's registry).
Township
Resources
Township resources accumulate slowly in-game, so this is handy. Storage caps still apply when Melvor loads the save, so very large values may be clamped. Buildings, worship and town layout remain in the Advanced editor.
Equipment
Slot Equipped item Qty
Equip any item your save already knows about (type for suggestions); × clears a slot. Spells, prayers, and equipment sets beyond the active one stay in the Advanced editor.
Advanced: raw JSON (everything)

Full save as JSON. ID-keyed data is tagged so re-export stays byte-perfect: { "#map": [[key, value], ...] }, { "#set": [...] }, { "#bytes": [...] }. Edit values, keep the tags. Changes here update the fields above when you click out of the box.
